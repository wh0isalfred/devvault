# devvault 

<!-- DevVault -->
DevVault is a platform that helps developers and companies securely store, organize, control, and retrieve sensitive information ("secrets") that applications need in order to function.

Examples of secrets include:

Database passwords
API keys
JWT signing keys
SSH private keys
Cloud credentials
Certificates
OAuth client secrets

Instead of placing these directly inside application code or .env files, organizations store them in DevVault and grant controlled access to the people and services that need them.

DevVault aims to answer four questions:

Where should secrets live?
Who should be allowed to access them?
How can we prove who accessed them and when?
How do we replace (rotate) secrets safely when needed?
